## Project Overview

The Paradise Nursery Shopping Application is a responsive single-page application (SPA) that simulates an online plant store. Users can:

- Browse a catalog of indoor plants
- View product descriptions and pricing
- Add plants to a shopping cart
- Increase or decrease item quantities
- Remove products from the cart
- View real-time cart totals
- Continue shopping or proceed to checkout

This project highlights my ability to build interactive web applications using modern JavaScript frameworks and industry-standard development tools.

- Welcoming homepage with branding and navigation

- Product images, descriptions, and prices

### 🛒 Shopping Cart
- Add to Cart functionality
- Quantity increase and decrease controls
- Item removal
- Automatic calculation of total cost and total number of items

### State Management
- Centralized cart state using Redux Toolkit reducers and actions

### Deployment
- Configured for production deployment with Vite and GitHub Pages
